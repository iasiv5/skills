# PUA 指令系统 & 本地配置

所有数据仅存储在本地，**不发送到任何外部服务**。

## 一、本地配置

配置文件：`~/.pua/config.json`

```json
{
  "flavor": "alibaba",
  "created_at": "ISO时间戳"
}
```

文件不存在时使用默认值（🟠 阿里味），不要求注册。

## 二、指令系统

所有指令在 `/pua` 命名空间下：

| 触发词 | 指令 |
|--------|------|
| `/pua` 或 `/pua help` | 查看所有指令 |
| `/pua flavor` | 味道切换 |
| `/pua kpi` | KPI 报告卡（本地统计） |

### /pua flavor — 味道切换

显示味道选择器，选择后写入 `~/.pua/config.json` 的 `flavor` 字段。

### /pua kpi — KPI 报告卡

分析当前会话工作内容，生成本地 KPI 报告：

```
┌─────────────────────────────────────────────────┐
│  📊 KPI 报告卡                                    │
│                                                  │
│  · 完成任务数：X                                  │
│  · [PUA生效 🔥] 次数：Y                           │
│  · 主动发现问题：Z 个                             │
│                                                  │
│  绩效等级：3.75 (S)                               │
└─────────────────────────────────────────────────┘
```

## 三、本地统计

统计数据存储在 `~/.pua/stats.json`，仅本地记录。格式：

```json
{"pua_count": 0, "sessions": 0, "last_updated": "ISO时间戳"}
```
