# 微信公众号 API 错误码参考

## Token 相关

| 错误码 | 含义 | 处理方式 |
|--------|------|---------|
| -1 | 系统繁忙 | 稍后重试 |
| 40001 | access_token 无效/过期 | 清缓存重试，或检查 APPID/APPSECRET |
| 40002 | grant_type 不合法 | 检查参数是否为 `client_credential` |
| 40003 | AppID 不合法 | 检查 app_id 是否正确 |
| 41001 | 缺少 access_token | 检查请求参数 |
| 42001 | access_token 已过期 | 重新获取 |
| 40125 | AppSecret 无效 | 检查 config.json 中的 app_secret |

## 权限相关

| 错误码 | 含义 | 处理方式 |
|--------|------|---------|
| 40164 | IP 不在白名单 | 公众号后台 → 设置与开发 → 基本配置 → IP 白名单 添加当前 IP |
| 48001 | API 未授权 | 公众号后台确认已获得草稿接口权限（需认证服务号） |
| 87009 | IP 不在白名单（同 40164） | 同上 |

## 资源相关

| 错误码 | 含义 | 处理方式 |
|--------|------|---------|
| 40004 | 媒体类型不合法 | 检查 type 参数 |
| 40009 | 图片大小超限 | 压缩至 2MB 以内 |
| 45009 | API 调用超限 | 稍后重试（每日 token 获取上限 2000 次） |

## 草稿相关

| 错误码 | 含义 | 处理方式 |
|--------|------|---------|
| 40007 | 不合法的媒体文件 id | 检查 thumb_media_id 是否有效 |
| 45064 | 创建草稿超过限制 | 单次最多 1 篇文章 |
| 40501 | 标题重复 | 修改标题后重试 |
